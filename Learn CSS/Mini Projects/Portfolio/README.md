# Raj Bhushan Dixit - Personal Portfolio

A modern, responsive personal portfolio website built to showcase projects, skills, and experience as a Full Stack Developer. The project is crafted using pure HTML5 and CSS3, emphasizing a clean, dark-themed UI, fluid typography, and semantic structure without relying on heavy JavaScript frameworks.

## ✨ Features

- **Fully Responsive Design:** Adapts fluidly to mobile phones, tablets, and large desktop screens utilizing CSS Flexbox and Grid.
- **Modern Dark Theme:** A carefully curated monochromatic purple/dark slate color palette defined via CSS custom properties (variables) for easy theming.
- **Micro-interactions & Animations:** Features a customized smooth typing animation in the hero section, soft button hover effects, and card scaling animations to create an engaging user experience.
- **Sticky Navigation:** A modern "floating pill" style navigation bar that remains highly visible and accessible while scrolling.
- **Working Contact Form:** The "Let's Connect" form is integrated with Formspree, allowing visitors to send emails directly from the site without a backend.
- **Clean Architecture:** Built with maintainability in mind—CSS variables are centralized, styling is modularized by section, and HTML tags are highly semantic.

## 🛠️ Technologies Used

- **HTML5:** For structured, semantic, and accessible content.
- **CSS3:** For all styling, layout (Flexbox/Grid), custom properties (variables), and keyframe animations.
- **Google Fonts:** Utilizing the `Inter` typeface for a clean, modern, and highly legible look.
- **Formspree:** For handling contact form submissions.

## 🚀 Getting Started

Because this project is built entirely with static HTML and CSS, no build steps or dependencies are required. 

1. **Clone or Download** the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in any modern web browser (Google Chrome, Firefox, Safari, Edge).

*Alternatively, you can open the project in an editor like VS Code and use the "Live Server" extension for hot-reloading as you make changes.*

## 🎨 Design System

This project adheres strictly to a predefined design system to ensure consistency. 
All colors, font sizings, spacing, and border-radii are stored in the `:root` pseudo-class in `styles.css` as variables. 

For an in-depth breakdown of the color codes, component styling rules, and typography hierarchy, please refer to the included [STYLEGUIDE.md](./STYLEGUIDE.md).
